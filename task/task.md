

# Спецификация архитектуры BIGA (Brain-Inspired Group Architecture)
## Версия 0.1 — Математическая модель

### 1. Введение: Почему мы строим это

Современные нейросети (трансформеры) добились впечатляющих результатов, но имеют фундаментальные ограничения:
- **Статичность знаний** — модель нельзя обновлять без переобучения
- **Отсутствие структуры** — все нейроны одинаковы, нет специализации
- **Энергонеэффективность** — каждый токен требует пересчета всех весов
- **Черный ящик** — мы не понимаем, где хранятся знания

Биологический мозг решает эти проблемы иначе:
- **Разные типы нейронов** (возбуждающие/тормозные) для регуляции
- **Группы (ядра)** со специализацией (зрение, слух, память)
- **Раздельное хранение** — память не в весах, а в связях
- **Постоянное обучение** — без эпох и батчей

BIGA — это попытка перенести эти принципы в математическую модель, пригодную для вычислений на современном железе.

---

### 2. Общая архитектура

Модель состоит из **пяти функциональных групп нейронов**, каждая из которых содержит два типа нейронов: возбуждающие (E) и тормозные (I).

#### 2.1 Группы

| Группа | Обозначение | Размерность (E/I) | Постоянная времени (τ_E, τ_I) | Функция |
|--------|-------------|-------------------|-------------------------------|---------|
| Сенсорная | S | 1000 / 200 | (1.0, 0.8) | Преобразует входные токены в активность |
| Синтаксическая | A1 | 2000 / 400 | (2.0, 1.5) | Обрабатывает структуру предложений |
| Семантическая | A2 | 2000 / 400 | (2.0, 1.5) | Обрабатывает значение слов |
| Памяти | M | 5000 / 1000 | (10.0, 8.0) | Хранит долговременные паттерны |
| Моторная | G | 1000 / 200 | (1.0, 0.8) | Генерирует выходные токены |

**Обоснование размерностей:**
- Память самая большая, потому что должна хранить много паттернов
- Синтаксис и семантика средние — достаточно для языка
- Сенсорная и моторная маленькие — только интерфейсы
- Тормозных нейронов ~20% от общего числа (биологическая пропорция)

**Обоснование постоянных времени:**
- Сенсорная/моторная быстрые — должны реагировать мгновенно
- Ассоциативные медленнее — интегрируют информацию
- Память самая медленная — хранит, а не обрабатывает

### 3. Математическая модель группы

#### 3.1 Состояние группы

Для каждой группы `g` в момент времени `t` мы храним два вектора:

```
y_g^E(t) ∈ ℝ^{N_g^E}  # активность возбуждающих нейронов
y_g^I(t) ∈ ℝ^{N_g^I}  # активность тормозных нейронов
```

Активность всегда неотрицательна (нейроны не могут иметь отрицательный firing rate).

#### 3.2 Динамика внутри группы

Изменение активности описывается системой дифференциальных уравнений:

```
τ_g^E · dy_g^E/dt = -y_g^E + f( W_g^{EE}·y_g^E + W_g^{EI}·y_g^I + b_g^E + I_g^E(t) )
τ_g^I · dy_g^I/dt = -y_g^I + f( W_g^{IE}·y_g^E + W_g^{II}·y_g^I + b_g^I + I_g^I(t) )
```

где:
- `W_g^{EE}, W_g^{EI}, W_g^{IE}, W_g^{II}` — матрицы рекуррентных связей внутри группы
- `b_g^E, b_g^I` — смещения (базовый уровень активности)
- `I_g^E(t), I_g^I(t)` — вход от других групп (определяется ниже)
- `f` — функция активации (ReLU, чтобы активность была неотрицательной)

**Смысл уравнений:**
- Первый член `-y` — затухание активности (если нет входа, нейрон "выключается")
- Второй член — сумма всех влияний (от других нейронов группы и извне)
- Деление на τ задает скорость реакции

#### 3.3 Ограничения на веса (биологический принцип)

Тип нейрона определяет знак его влияния:

- От возбуждающих нейронов (E) веса могут быть только **неотрицательными**
- От тормозных нейронов (I) веса могут быть только **неположительными**

Это реализуется через параметризацию:
```
W = sign(type) · softplus(V)  # где type = +1 для E, -1 для I
```
или через проекцию после обновления градиента.

### 4. Связи между группами

#### 4.1 Структура связей

Каждая группа получает вход от определенных других групп:

| Целевая группа | Источники входа |
|----------------|-----------------|
| S (сенсорная) | только внешний вход (нет межгрупповых) |
| A1 (синтаксис) | S, A1 (рекуррентно), A2, M |
| A2 (семантика) | S, A2 (рекуррентно), A1, M |
| M (память) | A1, A2 |
| G (моторная) | A1, A2, M, G (рекуррентно) |

#### 4.2 Математическое описание связей

Вход в группу `g` от группы `h`:

```
I_{h→g}^E = W_{h→g}^{EE}·y_h^E + W_{h→g}^{IE}·y_h^I
I_{h→g}^I = W_{h→g}^{EI}·y_h^E + W_{h→g}^{II}·y_h^I
```

где каждая матрица весов имеет размерность:
- `W_{h→g}^{EE}`: (N_g^E × N_h^E)
- `W_{h→g}^{IE}`: (N_g^E × N_h^I)
- и так далее.

**Ограничения:** те же — знак определяется типом нейрона-источника.

Полный вход в группу `g`:

```
I_g^E = Σ_{h ∈ sources(g)} I_{h→g}^E
I_g^I = Σ_{h ∈ sources(g)} I_{h→g}^I
```

### 5. Вход и выход системы

#### 5.1 Кодирование входных токенов (сенсорная группа)

Вход: последовательность токенов `x_1, x_2, ..., x_T` (целые числа от 0 до V-1).

Для каждого токена `x_t`:

1. Получаем эмбеддинг: `e_t = Embedding(x_t) ∈ ℝ^{d_emb}`
2. Добавляем позиционное кодирование: `e_t ← e_t + PE(t)`
3. Проецируем на размерность сенсорной группы: `p_t = W_in · e_t ∈ ℝ^{N_S^E}`

В момент времени `t` вход в сенсорную группу:

```
I_S^E(t) = p_t  # только для возбуждающих нейронов
I_S^I(t) = 0    # тормозные не получают прямого входа
```

#### 5.2 Декодирование выходных токенов (моторная группа)

Выходной токен на шаге `t` генерируется из активности моторной группы:

1. Берем среднее по возбуждающим нейронам: `a_t = mean(y_G^E(t)) ∈ ℝ^{N_G^E}`
2. Проецируем на размерность словаря: `logits_t = W_out · a_t ∈ ℝ^V`
3. Вероятность токена: `p(x|context) = softmax(logits_t)`

Обучаем модель предсказывать следующий токен `x_{t+1}`.

---

### 6. Динамика во времени (дискретизация)

Для численного решения мы используем метод Эйлера с шагом `dt = 0.1`:

```
y_new = y + dt · (dy/dt)
```

Для каждой группы:

```
y_g^E(t+dt) = y_g^E(t) + dt · ( -y_g^E(t) + f( total_input_g^E(t) + b_g^E ) ) / τ_g^E
y_g^I(t+dt) = y_g^I(t) + dt · ( -y_g^I(t) + f( total_input_g^I(t) + b_g^I ) ) / τ_g^I
```

где `total_input_g^E(t)` включает рекуррентные связи и вход от других групп.

После каждого шага применяем ReLU для сохранения неотрицательности.

### 7. Обучение

#### 7.1 Обучаемые параметры

- Все матрицы весов (с ограничениями)
- Смещения `b_g^E, b_g^I`
- Матрица эмбеддингов `Embedding`
- Входная проекция `W_in`
- Выходная проекция `W_out`

#### 7.2 Функция потерь

Основная потеря — кросс-энтропия для предсказания следующего токена:

```
L = - Σ_t log p(x_{t+1} | x_1...x_t)
```

#### 7.3 Поддержание ограничений

После каждого шага градиентного спуска применяем проекцию:

```
W = clamp(W, min=0)  # для весов от E-нейронов
W = clamp(W, max=0)  # для весов от I-нейронов
```

Это гарантирует, что знак влияния соответствует типу нейрона-источника.

#### 7.4 Инициализация

Все веса инициализируются малыми случайными значениями:
- Для E → * : положительные, из нормального распределения со std=0.01
- Для I → * : отрицательные, из нормального распределения со std=0.01 (со знаком минус)

Смещения инициализируются нулями.

---

### 8. Ожидаемые свойства

Из этой математической модели мы ожидаем:

1. **Стабильность** благодаря балансу E/I и затухающему члену `-y`
2. **Специализацию групп** из-за разной архитектуры связей
3. **Долговременную память** благодаря медленной динамике группы M
4. **Робастность к шуму** благодаря тормозным нейронам
5. **Возможность непрерывного обучения** без катастрофического забывания

---

### 9. Ограничения и допущения

На данном этапе мы сознательно упрощаем:

- Нет спайков — используем rate coding (частота импульсов)
- Нет астроцитов и нейромодуляторов
- Нет структурной пластичности (веса только меняются, связи не растут)
- Нет обучения во сне (отдельная консолидация)
- Все группы работают в одном временном масштабе (кроме разных τ)

Эти упрощения могут быть сняты в следующих версиях.

---

### 10. Заключение

BIGA v0.1 предлагает новый способ построения нейросетей, вдохновленный архитектурой мозга. Вместо однородных слоев — специализированные группы с разными типами нейронов. Вместо статичных весов — динамическая система с затуханием. Вместо черного ящика — интерпретируемая структура.

Математическая модель полностью определена и готова к реализации.
